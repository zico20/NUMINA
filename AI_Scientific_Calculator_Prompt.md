# 🧮 برومبت احترافي: تطبيق حاسبة علمية مدعومة بالذكاء الاصطناعي

> برومبت جاهز للاستخدام مع Claude Code، Cursor، GitHub Copilot، أو أي مساعد ذكاء اصطناعي للبرمجة.

---

## 🎯 المهمة الرئيسية (Mission Statement)

أريد منك بناء تطبيق **حاسبة علمية ذكية متكاملة** يعمل على منصتي **Android و iOS** بنفس الكود (Cross-platform)، يجمع بين قوة الحاسبة العلمية التقليدية وذكاء الـ AI الحديث. التطبيق يجب أن يكون احترافياً، سريعاً، جميل التصميم، وسهل الاستخدام لكل الفئات (طلاب، مهندسين، مستخدمين عاديين).

---

## 📱 المواصفات التقنية الأساسية

### المنصات المستهدفة
- **Android** (الإصدار 8.0 / API 26 فما فوق)
- **iOS** (الإصدار 13.0 فما فوق)

### التقنيات المقترحة (Tech Stack)
| الطبقة | التقنية الموصى بها | بدائل |
|--------|-------------------|--------|
| Frontend | **Flutter (Dart)** | React Native (TypeScript) |
| State Management | Riverpod أو Bloc | Redux / Provider |
| Backend (اختياري) | Node.js + Express أو Firebase Functions | Python FastAPI |
| AI Model | **OpenAI GPT-4o (Vision)** | GPT-5 عند توفره |
| Math Engine | `math_expressions` + `flutter_math_fork` | Math.js |
| Database (محلي) | Hive أو SQLite (sqflite) | Isar |
| Image Processing | `image_picker` + `image_cropper` | - |
| Charts | `fl_chart` أو `syncfusion_flutter_charts` | - |
| LaTeX Rendering | `flutter_math_fork` | - |

### اللغات المدعومة
- **العربية** (RTL كامل) — افتراضية حسب لغة الجهاز
- **English** (LTR)
- استخدام `flutter_localizations` و `intl` للترجمة الديناميكية

## ⚙️ الميزات الأساسية للحاسبة العلمية

### 1. العمليات الحسابية الأساسية
- الجمع، الطرح، الضرب، القسمة
- النسبة المئوية، الكسور، القيمة المطلقة
- الأقواس المتداخلة
- الأرقام السالبة والموجبة

### 2. الدوال العلمية المتقدمة
- **المثلثية**: `sin`, `cos`, `tan`, `sec`, `csc`, `cot` + معكوساتها (`asin`, `acos`, ...)
- **الزوايا**: تبديل بين Degrees / Radians / Gradians
- **اللوغاريتمات**: `log`, `ln`, `log_base_n`
- **الأسس والجذور**: `x²`, `x³`, `xⁿ`, `√`, `∛`, `ⁿ√`
- **الثوابت**: `π`, `e`, `φ` (النسبة الذهبية)
- **المضروب** (Factorial): `n!`
- **التركيب والتباديل**: `nCr`, `nPr`
- **الأرقام المركبة** (Complex Numbers)
- **التحويل بين الأنظمة العددية**: Decimal, Binary, Octal, Hexadecimal

### 3. حسابات متقدمة
- **التفاضل والتكامل** الرمزي (Symbolic Differentiation & Integration)
- **حل المعادلات** (خطية، تربيعية، تكعيبية، نظام معادلات)
- **المتباينات**
- **الإحصاء**: المتوسط، الوسيط، المنوال، الانحراف المعياري، التباين، الانحدار الخطي
- **المصفوفات** (Matrices): الجمع، الضرب، التحويل (Transpose)، المقلوب (Inverse)، المحدد (Determinant)، القيم الذاتية (Eigenvalues)

## 🤖 ميزة تشخيص الصور بالذكاء الاصطناعي (الميزة الرئيسية)

### التدفق الكامل (User Flow)

```
1. المستخدم يضغط زر الكاميرا 📸
   ↓
2. خيارات: التقاط صورة جديدة / اختيار من المعرض / لصق صورة
   ↓
3. اقتصاص وتحسين الصورة (Crop + Enhance)
   ↓
4. إرسال الصورة إلى GPT-4o Vision API
   ↓
5. الـ AI يستخرج المعادلة بصيغة LaTeX
   ↓
6. عرض المعادلة المُستخرجة للمستخدم للتأكيد/التعديل
   ↓
7. اختيار نوع الحل:
   ┌─────────────────────┬────────────────────────┐
   │  ⚡ حل سريع          │  📚 حل تفصيلي           │
   │  (الإجابة فقط)        │  (خطوة بخطوة + شرح)     │
   └─────────────────────┴────────────────────────┘
   ↓
8. عرض الحل بتنسيق LaTeX جميل + إمكانية المشاركة/الحفظ
```

### تفاصيل تكامل OpenAI GPT-4o Vision

```dart
// مثال على هيكل الطلب (Request Structure)
{
  "model": "gpt-4o",
  "messages": [
    {
      "role": "system",
      "content": "أنت مساعد رياضيات خبير. مهمتك: 
                  1) استخراج المعادلة من الصورة بصيغة LaTeX 
                  2) حلها بدقة 
                  3) تقديم الحل بتنسيق JSON منظم"
    },
    {
      "role": "user",
      "content": [
        {"type": "text", "text": "حل هذه المعادلة: [نوع الحل]"},
        {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,..."}}
      ]
    }
  ],
  "response_format": { "type": "json_object" }
}
```

### معالجة الأخطاء (Error Handling)
- فشل قراءة الصورة → اقتراح إعادة التصوير بإضاءة أفضل
- معادلة غير واضحة → السماح للمستخدم بالتعديل اليدوي على LaTeX المُستخرج
- فشل الاتصال بالـ API → عرض رسالة واضحة + إمكانية إعادة المحاولة
- معالجة Rate Limiting و Token Limits

### الأمان (Security)
- **عدم تخزين مفتاح OpenAI API في كود التطبيق مطلقاً**
- استخدام Backend وسيط (Proxy Server) لإخفاء المفتاح
- تشفير الصور أثناء النقل (HTTPS)
- عدم حفظ صور المستخدمين على السيرفر بعد المعالجة (Privacy First)

## ✨ الميزات الإضافية

### 1. 📊 الرسم البياني (Graphing)
- رسم الدوال 2D بشكل تفاعلي
- إمكانية رسم أكثر من دالة معاً (مع ألوان مختلفة)
- التكبير/التصغير (Pinch to Zoom)
- التنقل (Pan)
- إظهار نقاط التقاطع، القمم، والأصفار
- تصدير الرسم البياني كصورة PNG
- إدخال الدالة يدوياً أو من نتيجة الـ AI

### 2. 📝 سجل الحسابات (History)
- حفظ تلقائي لكل العمليات
- البحث والفلترة (بالتاريخ، النوع، الكلمات المفتاحية)
- إمكانية تثبيت العمليات المهمة (Pin)
- إعادة استخدام النتائج السابقة بضغطة واحدة
- مزامنة اختيارية عبر السحابة (Firebase) للمستخدمين المسجلين
- إمكانية حذف السجل كاملاً أو انتقائياً

### 3. 🌗 الوضع الليلي / النهاري
- ثلاثة أوضاع: تلقائي (حسب الجهاز) / فاتح / داكن
- استخدام Material You (Android) و Dynamic Color
- ألوان متناسقة بدقة في الوضعين
- انتقال سلس بين الأوضاع (Animation)

### 4. 🔄 محول الوحدات (Unit Converter)
أنواع التحويل:
- **الطول**: متر، كم، ميل، قدم، بوصة، يارد
- **الوزن**: غرام، كغم، طن، باوند، أونصة
- **الحجم**: لتر، ملم، غالون، كوب
- **درجة الحرارة**: مئوية، فهرنهايت، كلفن
- **الوقت**: ثانية، دقيقة، ساعة، يوم، أسبوع، سنة
- **الزاوية**: درجة، راديان، غراديان
- **السرعة**: كم/س، م/ث، ميل/س، عقدة
- **الضغط**: باسكال، بار، PSI
- **الطاقة**: جول، كالوري، كيلوواط ساعة
- **العملات** (Live Rates): استخدام API مجاني مثل `exchangerate-api.com`

### 5. 📤 المشاركة والتصدير
- تصدير الحلول كـ **PDF** بتنسيق احترافي (يحتوي على LaTeX مرسوم)
- مشاركة كصورة عالية الجودة (PNG)
- نسخ نص LaTeX إلى الحافظة
- مشاركة مباشرة عبر: WhatsApp, Telegram, Email, إلخ
- حفظ النتائج محلياً في تطبيق Files

6. **شاشة محول الوحدات**: تصميم بسيط وسهل

```
├── core/                    # الأدوات المشتركة
│   ├── constants/
│   ├── errors/

│   ├── calculator/
│   │   ├── data/           # API, Local DB
│   │   ├── domain/         # Entities, Use Cases
│   │   └── presentation/   # UI, State
│   ├── ai_solver/
│   ├── graphing/
│   ├── history/
│   ├── unit_converter/
│   └── settings/
├── shared/                  # المكونات المشتركة
```

### Backend (Proxy لـ OpenAI):
```
backend/
├── src/
│   ├── controllers/
│   ├── middlewares/      # Auth, Rate Limiting
│   ├── services/         # OpenAI Service
│   └── routes/
```

## 🛡️ الأمان والخصوصية

- **لا تخزين**: عدم حفظ صور المستخدمين على السيرفر

- **التحقق من المدخلات**: Validation شامل
- **عدم تتبع المستخدم** بدون موافقة صريحة
